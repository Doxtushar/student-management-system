# Student Management System

A Full Stack Student Management System built using **Angular 16**, **Spring Boot**, and **MySQL**. The application allows users to securely manage student records through a modern, responsive web interface.

---

## Features

- Secure User Authentication
- Add Student
- View Student List
- Update Student Details
- Delete Student
- Search Students
- Responsive UI using Angular Material
- RESTful API Integration
- Form Validation
- Swagger API Documentation
- Cloud Deployment

## Installation

### Clone Repository

```bash
git clone https://github.com/Doxtushar/student-management-system.git
```

---

### Backend

```bash
cd practice
mvn clean install
mvn spring-boot:run
```

---

### Frontend

```bash
cd studentManagement
npm install
ng serve
```

Open

```
http://localhost:4200
```
